## Installation
### Step: (copy the step)
1. `git clone https://github.com/shiazza/perpustakaan-sekolah.git`
2. `composer install`
3. `cp .env.example .env`
4. `php artisan key:generate`
5. `php artisan migrate`
6. `php artisan db:seed`
7. `php artisan serve`
